# APPOLOC Website

Modular wall systems — salespage with contact form.

## Setup

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Push to GitHub
2. Connect repo to Vercel
3. Add environment variable: `RESEND_API_KEY`
4. Deploy

## Structure

- `src/App.jsx` — Complete salespage (DE/EN)
- `api/send-inquiry.js` — Contact form API (Resend)
- `public/images/` — Optimized product images
